# 🌸 Little Love Studio

**Little Love Studio** is a beautiful interactive web application that allows users to create and send a **virtual bouquet filled with emotions**.
Users can choose flowers, set a mood, write heartfelt messages, and send a personalized digital surprise.

The platform focuses on **emotional storytelling through design**, combining animated bouquets, handwritten letters, and aesthetic UI to create memorable experiences.

---

# ✨ Features

### 💐 Bouquet Builder

* Choose from multiple hand-illustrated flowers
* Automatically arranged bouquet layout
* Add greenery and decorative elements
* Up to **10 flowers per bouquet**

### 🎭 Mood Selection

Select the emotional tone of your bouquet:

* Romantic
* Playful
* Tender
* Passionate
* Grateful
* Nostalgic

### 💌 Write a Letter

* Add **To / From** fields
* Write a personalized message
* Beautiful handwritten-style preview
* Live preview of the letter card

### 🎁 Animated Surprise

* A gift-style reveal animation
* Beautiful bouquet opening experience
* Emotional storytelling through UI

### 🔗 Share Anywhere

* Generate a bouquet
* Share the surprise with someone special

# 🚀 Getting Started

Clone the repository:

```
git clone https://github.com/priyanshikothari10/littlelovestudio.git
```

Navigate to the project folder:

```
cd littlelovestudio
```

Install dependencies:

```
npm install
```

Run the development server:

```
npm run dev
```

Open in browser:

```
http://localhost:3000
```
